# Steam Badge Helper

Tampermonkey 用户脚本，在 Steam 徽章页面扫描有未完成进度的游戏徽章，批量查询卡牌价格，计算卡牌套装成本，一键跳转 Steam 批量购买页面并自动填入数量和价格。

> **v1.0.0**：完整扫描 → 购买流程。自动跳转 multibuy 页面并填入每张卡牌的需求数量和价格（最低价 + 上浮），用户只需确认下单。

## 功能

- 自动扫描徽章页面，找出有未完成进度的游戏
- 批量查询卡牌市场价格（含最低价与中位数）
- 按单套卡牌价格上限筛选
- 满级价格估算（最低价 + 中位数混合，绿色=近期有成交准，灰色=无成交不准）
- 点击 **购买** → 自动跳转 multibuy 页面 → 数量 + 价格自动填入
- multibuy 页面 **恢复默认价格** 按钮 → 一键刷新重置
- 黑名单管理：查询游戏名称 + 添加/移除 AppID
- 点击结果行空白区域 → 新标签页打开对应徽章页面

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 打开脚本的 GitHub raw 链接，Tampermonkey 会自动弹出安装提示：
   ```
   https://github.com/SpaceSyt/Steam-Badge-Helper/raw/master/steam-badge-helper.user.js
   ```
3. 安装后打开任意 Steam 徽章页 (`steamcommunity.com/id/xxx/badges/`)，页面右侧会出现 **Steam Badge Helper** 按钮

## 使用说明

1. 点击入口按钮打开助手面板
2. 设置 **单套卡牌价格上限**、**购买卡牌逻辑**、**价格上浮**
3. 点击 **开始扫描**，等待扫描完成
4. 点击结果行中的 **购买** → multibuy 页面自动打开并填好数量和价格 → 确认下单

### 购买卡牌逻辑

| 模式 | 说明 |
|------|------|
| 补全单套 | 只买当前缺失的卡牌，每样 1 张 |
| 补全五套 | 每张卡补到 5 张（满级所需） |
| 购买单套 | 每样卡牌各买 1 张（不管是否拥有） |
| 购买五套 | 每样卡牌各买 5 张（合成满级） |

## 配置项

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 单套卡牌价格上限 | ¥5 | 全套卡牌最低价之和的上限 |
| 购买卡牌逻辑 | 补全五套 | multibuy qty 计算方式 |
| 价格上浮 | ¥0.10 | 填入价格 = 最低价 + 上浮，提高购买成功率 |
| 最大徽章页数 | 1 | 扫描的最大页数 |
| 包含掉落 | 关 | 是否包含仍有掉落的游戏 |
| 请求间隔 | 1200ms | 每次请求间隔 |
| 批暂停 | 20次/15s | 防止限流 |

## 注意事项

- 仅支持人民币 (CNY)
- 满级价格为估算值，成交量低时可能不准
- 不支持闪卡

## Steam 请求速率限制

Steam 市场 API 的具体速率限制官方并未公开。当前脚本的默认请求间隔（1200ms，每20次暂停15秒）可以稳定运行约 1-2 小时。但如果短时间内发起过多请求，Steam 可能会**暂时封锁 IP 地址**。

**如果你发现按 F12 打开浏览器开发者工具 → Network 标签页，所有 `priceoverview` 请求都返回 429 状态码，说明 IP 已被暂时封锁**。解决方法：

- 等待几个小时让封锁自动解除
- 更换 IP 地址（例如使用 VPN 切换节点）

脚本扫描过程中如遇到限流，会在状态栏显示"限流冷却中"，并自动等待后重试。但如果持续被 429 拒绝，建议停止扫描并等待。
