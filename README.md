# Steam Badge Helper

Tampermonkey 用户脚本，在 Steam 徽章页面扫描有未完成进度的游戏徽章，批量查询卡牌价格，计算卡牌套装成本，一键跳转 Steam 批量购买页面并自动填入数量和价格。

> **v1.2.3**：完整扫描 → 购买流程。自动跳转 multibuy 页面并按当前徽章等级填入满级所需数量和价格（最低价 + 上浮），用户只需确认下单。

## 功能

- 自动扫描徽章页面，找出有未完成进度的游戏
- 批量查询卡牌市场价格（含最低价与中位数）
- 满级价格估算（绿色灰色红色标示成交量参考度）
- 一键跳转 multibuy 批量购买
  - 自动填入数量和价格（最低价 + 上浮）
  - 四种购买逻辑可选：补全单套 / 补至五级 / 购买单套 / 购买五套
  - multibuy 页恢复默认价格按钮
- 黑名单管理
  - 查询游戏名称 + 添加/移除 AppID
  - 自动黑名单：扫描时超限自动加入（可选开关）
- 点击结果行空白区域 → 新标签页打开徽章页面

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 打开脚本的 GitHub raw 链接，Tampermonkey 会自动弹出安装提示：
   ```
   https://github.com/SpaceSyt/Steam-Badge-Helper/raw/master/steam-badge-helper.user.js
   ```
3. 安装后打开任意 Steam 徽章页 (`steamcommunity.com/id/xxx/badges/`)，页面右侧会出现 **Steam Badge Helper** 按钮

## 使用说明

1. 点击入口按钮打开助手面板
2. 设置 **单套卡牌价格上限**、**购买卡牌逻辑**、**价格上浮**
3. 点击 **开始扫描**，等待扫描完成
4. 点击结果行中的 **购买** → multibuy 页面自动打开并填好数量和价格 → 确认下单

## 配置项

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 单套卡牌价格上限 | ¥5 | 全套卡牌最低价之和的上限 |
| 购买卡牌逻辑 | 补至五级 | multibuy qty 计算方式；按当前徽章等级补至五级 |
| 价格上浮 | ¥0.10 | 填入价格 = 最低价 + 上浮 |
| 最大徽章页数 | 1 | 扫描的最大页数 |
| 包含掉落 | 关 | 是否包含仍有掉落的游戏 |
| 请求间隔 | 450ms | 每次 `priceoverview` 请求间隔 |
| 批暂停 | 20次/45s | 每 20 次 `priceoverview` 后主动冷却 |

## 注意事项

- 仅支持人民币 (CNY)
- 满级价格为估算值，成交量低时可能不准（绿色 >1 / 灰色 =1 / 红色 =0）
- 不支持闪卡

## Steam 请求速率限制

Steam 市场 API 的具体速率限制官方并未公开。当前脚本默认每次 `priceoverview` 请求间隔 450ms，并在每 20 次请求后主动暂停 45 秒。如果仍收到 429，脚本会额外暂停 45 秒后重试，作为被动限流保护。

如果按 F12 打开开发者工具，所有 `priceoverview` 请求都返回 429，说明 IP 可能已被暂时限制。解决方法：等待几小时或更换 IP。扫描中连续 5 次 429 后日志区会显示提醒。
